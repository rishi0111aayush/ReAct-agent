🤖 How Boozo.ai Works

A simple step-by-step guide — explained like you're in class 7!

💬
Step 1
You Type a Question
You open the chat and type something like "What is Python?" or "Show me a chart of top languages". You press Enter. That's where the magic starts!
📦
Step 2
Browser Sends Your Message to the Server
Your browser (Chrome, Edge etc.) packages your message and sends it over the internet to our FastAPI server running on your computer at localhost:8000. Think of it like sending a letter to a post office.
🧠
Step 3
The Agent Starts Thinking (ReAct Loop)
The server hands your question to Boozo.ai's brain — the ReAct agent. ReAct stands for Reason + Act. It means: first think, then do something, then think again. Like how you solve a maths problem step by step.
⚡
Step 4
It Asks a Giant AI Brain (LLM)
The agent sends your question to a Large Language Model (LLM) — a super-smart AI trained on millions of books and websites. We try them in order until one works:

Step 5
The LLM Reads and Plans
The LLM reads your question plus a special instruction that says: "Think step by step. If you need more info, use a tool. Otherwise, give the final answer." It writes back a Thought (what it plans to do).
DECISION
🤔 Does it need
a TOOL?
✅ YES — Use a Tool
Runs a Tool
The agent picks one of its tools:

🔍 web_search — searches DuckDuckGo
💻 code_exec — runs Python code
🧮 calculator — does maths
📥 rag_ingest — stores a document
🔎 rag_search — finds stored info
📊 plot — draws a chart

The tool runs and sends back an Observation (the result). Then the loop goes back to Step 3 and thinks again!
🏁 NO — Ready to Answer
Writes Final Answer
The agent has enough info to answer you directly. It writes "Final Answer: …" and stops the loop.

The answer is streamed back to your screen one letter at a time — that's why it appears like it's typing!
📡
Step 6
Answer Streams Back to Your Screen
The server sends the answer back using SSE (Server-Sent Events). Think of it like a live TV broadcast — data flows continuously from the server to your browser. That's why you see the text appearing letter by letter instead of all at once.
🖥️
Step 7
Right Panel Updates Live
While you wait, the right panel shows you everything happening behind the scenes: the agent's thoughts, which tools it used, how long each step took (latency), and how much of the context window was used. It's like watching the agent's brain through a glass window!
✅
Step 8
Done! Answer Displayed
The final answer appears in the chat bubble. If the agent made a chart, it draws it right there in the chat. Your question + the agent's answer are saved in history so the agent remembers the conversation for next time.

📚 Key Words Explained Simply

LLM — A giant AI brain trained on the whole internet
ReAct Loop — Think → Act → Think again, repeat
Tool — A special action the agent can run (search, code, etc.)
SSE — Live data stream from server to browser
FastAPI — The post office that handles your messages
RAG — Agent's notebook where it stores & finds documents
Scratchpad — Agent's rough work (Thoughts + Observations)
Fallback — If Plan A fails, try Plan B automatically
